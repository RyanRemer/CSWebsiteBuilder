Installing and Setting Up Git

Follow the instructions below to set up Git on your computer.

  1. Go to https://git-scm.com/download/.
  2. Download git by selecting your operating system

  1. Install git with the default options (basically just press “Next” until you are done).

  1. Check if you successfully installed git by typing git --version in your terminal

  1. If you see a version number, it means git was installed.

  1. Type the following commands from the command line to finish setting up git:

  1. git config --global user.name “<First Last>”
  2. git config --global user.email “<email>”

  1. Type git config --list and check to see that your name and email are in the user.name and user.email fields.
