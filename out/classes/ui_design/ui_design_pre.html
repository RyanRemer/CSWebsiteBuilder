UI Design Basics

Your pre-class assignment is to first read and watch the articles and videos about UI Design, then to explore the resources at the end of the page.

Resources

Below is a list of resources that you might find useful for UI design. You can be quizzed on these, so make sure you know what each website is for.

Material Design

A great website that teaches a standardized design.

https://material.io/design/

Also has great tools for picking colors

https://material.io/design/color/the-color-system.html#tools-for-picking-colors

https://material.io/tools/color/#!/?view.left=0&view.right=0&primary.color=1976D2

Font Awesome

A website with free icons for your applications.

https://fontawesome.com/

Google Fonts

A website of different fonts that shows how some fonts work well together.

https://fonts.google.com/

Lorem Ipsum

A website that creates filler text for your designs.

https://www.lipsum.com/
